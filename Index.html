HOT KEY

Two-key combination

Знайти на сторінці / Find On Page Ctrl + F
Відмінити, крок назад / Undo Ctrl + Z
Копіювати / Copy Ctrl + C
Вирізати / Cut Ctrl + X
Вставитит / Paste Ctrl + V
Вибрати все / Select All Ctrl + A
Додати до наступного збігу / Add Selection To Next Find Match Ctrl + D
Зберегти файл / File: Save Ctrl + S
Лінія виступів / Outdent Line Ctrl + " [ "
Відступ лінії / Indent Line Ctrl + " ] "
Відступ під рядком / Indent Line Below Ctrl + Enter
Перемістити рядок вниз / Move Line Down Alt + DownArrow
Перемістити рядок Вверх / Move Line Up Alt + UpArrow

Three-key combination

Скопіювати рядок вниз / Copy Line Down Shift + Alt + DownArrow
Скопіювати рядок вгору/Copy Line Up Shift + Alt + UpArrow
Відступ над рядком / Indent Line Above Ctrl + Shift + Enter
Додати курсор Нижче / Add Cursor Belove Ctrl + Alt + DownArrow
Додати курсор вище/Add Cursor Above Ctrl + Alt + UpArrow
